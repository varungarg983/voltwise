# VoltWise

Static website app.

## Files
- index.html
- style.css
- script.js
- assets/

## Deploy
1. Keep the folder structure unchanged.
2. Upload or host the folder contents on any static hosting platform.
3. Since this is a static site, there is no build command.

## Notes
- Uses Chart.js and Mermaid from CDN.
- Petrol fetch attempts the official MBIE CSV. If browser CORS blocks it, the manual/default value stays in place.
